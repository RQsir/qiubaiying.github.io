Ruby 安装分3中途径：

  1. 官方下载源码，编译make，然后make install，优点是可以保持版本最新
  2. 通过系统自带apt 安装，sudo apt install -y ruby-full ，简单快捷，但一般版本偏旧
  3. 通过第三方包管理器安装，这里选择RVM
